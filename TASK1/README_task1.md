# 🤖 SmartBot - Rule-Based Chatbot

## 📌 Project Overview

SmartBot is a simple Rule-Based Chatbot developed using Python. The chatbot interacts with users through text-based conversations and responds according to predefined rules and patterns.

Unlike AI-powered chatbots, SmartBot does not learn from data. Instead, it uses conditional statements (`if-elif-else`) and keyword matching techniques to identify user intentions and provide appropriate responses.

This project demonstrates the basic concepts of:

- Natural Language Processing (NLP)
- Pattern Matching
- Conversation Flow Design
- User Interaction
- Decision Making in Python

---

## 🎯 Objective

The main objective of this project is to create an interactive chatbot capable of:

- Understanding user queries through keyword detection
- Responding with predefined answers
- Maintaining a conversational flow
- Demonstrating fundamental chatbot architecture

## 🧠 Working Principle

The chatbot follows a rule-based approach.

### Step 1:
User enters a message.

### Step 2:
Input is converted to lowercase.

### Step 3:
Keywords are checked using pattern matching.

### Step 4:
Matching condition is executed.

### Step 5:
Bot displays the corresponding response.

### Step 6:
Conversation continues until user enters an exit command.
