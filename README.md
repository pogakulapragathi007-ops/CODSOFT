# ΏθΥΝ Repository Overview

This repository contains three Artificial Intelligence and Machine Learning projects developed as part of an AI/ML Internship program. The projects demonstrate fundamental concepts of Machine Learning, Artificial Intelligence, Natural Language Processing, Recommendation Systems, and Game Theory.

# ΏθΟψ Internship Tasks

## Task 1: Rule-Based Chatbot

A simple chatbot developed using Python that responds to user inputs based on predefined rules and keyword matching techniques.

Key Features

- Greeting recognition
- Educational assistance
- Study guidance
- Motivational quotes
- Date and time information
- Help system
- Exit functionality

Concepts Used

- Natural Language Processing (Basic)
- Pattern Matching
- Conditional Statements
- Conversation Flow Design

---

## Task 2: Tic-Tac-Toe AI

An intelligent Tic-Tac-Toe game where a human player competes against an AI agent. The AI uses the Minimax Algorithm to make optimal decisions and provide challenging gameplay.

Key Features

- Human vs AI gameplay
- Intelligent move selection
- Winner detection
- Draw detection
- Input validation
- Unbeatable AI logic

Concepts Used

- Artificial Intelligence
- Minimax Algorithm
- Game Theory
- Search Algorithms
- Decision Trees

---

## Task 3: Recommendation System

A content-based recommendation system that suggests movies to users based on similarity analysis. The system uses cosine similarity to identify related items and generate recommendations.

Key Features

- Movie recommendations
- Similarity analysis
- Fast recommendation generation
- User-friendly interface
- Content-based filtering

Concepts Used

- Recommendation Systems
- Machine Learning Fundamentals
- Content-Based Filtering
- Cosine Similarity
- Data Analysis
